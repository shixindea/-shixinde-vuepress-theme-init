开发 袁果锅

## 快速开始

### 安装包 `npm i @shixinde/vuepress-theme-init`

```
npm i @shixinde/vuepress-theme-init
```

### 找到`node_modules`中的`@shixinde` 将文件夹中的 `vuepress-theme-init`文件夹提取出来

### 执行命令安装依赖

```
npm i 或者 pnpm i
```

### 运行
```
npm run docs:dev 或者 pnpm docs:dev
```
